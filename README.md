## <samp>❯ 01 — The premise</samp>

I'm a **designer who codes**. I take a product from the first sketch to the last line of code — **no handoff, no lost intent**, just one continuous craft.

- 🎨 **Design + engineering in one head** — the intent never gets translated away
- 🧩 **Systems over screens** — tokens, components and motion that scale
- 🚀 **Performance is a feature** — budgets, sub-second interactions, buttery motion
- ✨ I obsess over **the last 5%** everyone else skips: spacing, easing, the feel
